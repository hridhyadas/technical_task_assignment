# Orange PR Technical Task Assignment 

This repository contains the solution for the Orange PR Technical Task Assignment. The solution is implemented as a responsive landing page using HTML, CSS, and JavaScript.

## Project Overview

The project is a modern, responsive landing page for a mobile telecommunications company, likely "Orange". It features a premium design with smooth scrolling, interactive elements, and a sticky header. The design is optimized for both desktop and mobile devices.

## Features

- **Hero Section**: A prominent hero section with a compelling headline, subtext, and dual call-to-action buttons ("Explore Now" and "Buy Now").
- **Sticky Header**: A fixed header that remains visible while scrolling, containing the company logo, navigation links, and action buttons.
- **Animated Logo**: A looping SVG animation for the "Orange" logo in the top-left corner.
- **Responsive Navigation**: A hamburger menu for mobile devices that slides out smoothly.
- **Interactive Scroll Animations**: Elements fade in and slide up as the user scrolls down the page.
- **Grid Layouts**: Features a "Magic Grid" section with an interactive mosaic of images that highlight different service plans or features.
- **Modern Typography**: Utilizes Google Fonts (Poppins andOutfit) for a clean and professional look.
- **Smooth Animations**: All interactive elements use CSS transitions for a premium feel.

## Technology Stack

- **HTML5**: Semantic markup for the page structure.
- **CSS3**: Custom properties (variables), flexbox, grid, and keyframe animations.
- **JavaScript**: For mobile menu functionality and scroll animations.
- **SVG**: Animated logo and icons.

## Directory Structure

- **`index.html`**: The main entry point of the application, containing all HTML content.
- **`assets/`**: Contains all static assets:
    - **`css/style.css`**: Main stylesheet with all CSS rules and animations.
    - **`js/main.js`**: JavaScript for interactive features.
    - **`images/`**: All images used in the project.
        - `logo.svg`: Animated Orange logo.
        - `header-img.jpg`: Hero background image.
        - `mosaic-grid/`: Images for the grid layout.

## Local Development

To run this project locally, simply open the `index.html` file in your web browser.

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd Orange_PR_Task
    ```
2.  Open `index.html` in your browser.
